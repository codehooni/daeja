# 출시 노트 (Release Notes)

## 버전 [미정] - 2026년 3월

### 🎉 새로운 기능

#### 발렛 주차 예약 시스템
- **날짜 기반 요금 계산**: 입차/출차 시간을 기준으로 자동 요금 계산
  - 발렛 요금: 기본 20,000원
  - 일일 주차 요금: 5,000원/일
  - 사용 일수에 따른 자동 계산 (당일 = 1일)
- **예약 시 요금 저장**: 예약 시점의 요금 정책을 저장하여 이후 가격 변동에도 원래 가격 유지
- **상세 요금 내역**: 예약 상세 화면에서 발렛 요금, 주차 요금, 전체 요금 구분 표시

#### 푸시 알림
- **예약 상태 알림**: 예약 승인, 거절, 차량 준비 완료 등 실시간 푸시 알림
- **알림 로깅 시스템**: 알림 전송/수신 내역 상세 로그 기록
- **딥링크 지원**: 알림 클릭 시 해당 예약 화면으로 자동 이동

#### 전화 걸기 기능
- **담당 직원 연락**: 예약 상세 화면에서 담당 직원에게 바로 전화 걸기

#### 차량 타입별 이미지
- 5가지 차량 타입 이미지 추가 (Compact, Sedan, SUV, Ioniq 6, Staria)
- 예약 화면에서 차량별 맞춤 이미지 표시

### ✨ 개선사항

#### UI/UX 개선
- **발렛 주차 시트 디자인 개선**: 더 직관적이고 깔끔한 인터페이스
- **예약 완료 화면 추가**: 예약 성공 시 안내 정보 및 다음 단계 안내
- **예약 상세 화면 리팩토링**:
  - 가독성 향상을 위한 레이아웃 개선
  - 요금 정보 섹션 추가
  - 정보 그룹화 및 시각적 계층 구조 개선
- **검색 결과 바텀시트 분리**: 검색 결과에서 주차장 카드 탭 시 지도 마커와 동일하게 발렛/일반 주차장 바텀시트 구분 표시
  - 발렛 주차장 → PREMIUM VALET 배지, 보라색 테마, 예약하기 버튼이 포함된 발렛 전용 시트
  - 일반 주차장 → 기존 일반 주차장 시트

#### 발렛 마커 개선
- **Glow 효과 추가**: 발렛 주차장 마커에 시각적 강조 효과
- 일반 주차장과 발렛 주차장 구분 용이

#### 예약 안내 정보
- 발렛 서비스 이용 방법 상세 안내
- 예약 시 주의사항 표시
- 예상 요금 미리 보기

### 🐛 버그 수정

#### 예약 시스템
- Firestore 데이터 저장/읽기 오류 수정
- 타입 캐스팅 에러 수정 (String/int 안전 변환)
- 예약 생성 시 필드 누락 문제 해결

#### 데이터 동기화
- 예약 정보 실시간 업데이트 개선
- Firestore Timestamp 변환 로직 안정화

### 🔧 기술적 개선

#### 새로운 서비스
- `PriceService`: 요금 계산 로직 중앙화
- `NotificationService`: 푸시 알림 관리
- `PhoneCallService`: 전화 걸기 기능

#### 아키텍처
- Clean Architecture 원칙 준수
- Domain/Data/Presentation 계층 명확한 분리
- Entity-Model 매핑 패턴 일관성 유지

#### 데이터 모델 확장
- `Reservation` 모델에 요금 필드 추가 (`valetFee`, `dailyParkingFee`)
- `ParkingLot` 모델에 가격 정책 필드 추가 (`basePrice`, `unitPrice`)
- `PrivateParking` 엔티티 확장

### 📦 의존성 업데이트
- Firebase 푸시 알림 관련 패키지 추가
- URL Launcher 패키지 추가 (전화 걸기)

발렛 요금: ₩20,000

```

주차 요금 (3일): ₩15,000
전체 요금: ₩35,000
```
